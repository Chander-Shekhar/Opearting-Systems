To compile the CAS :
g++ SrcAssgn2-cas-CS17BTECH11011.cpp -o cas -lpthread

To compile the TAS :
g++ SrcAssgn2-tas-CS17BTECH11011.cpp -o tas -lpthread

To compile the CAS-bounded :
g++ SrcAssgn2-cas-bounded-CS17BTECH11011.cpp -o cas-bounded -lpthread

To run the CAS :
./cas < inp-params.txt

To run the TAs :
./tas < inp-params.txt

To run the CAS-bounded :
./cas-bounded < inp-params.txt

Note: keep the inp-params.txt file in the same folder as these. You need to pass inp-params.txt as a argument.
